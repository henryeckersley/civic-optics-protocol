COP-1 Semi-Automated Analyzer

This version actually analyzes content. Paste a URL or paste the page text/HTML. The tool extracts evidence, suggests scores for all five categories, and lets you override them before exporting a report.

Because this is a static GitHub Pages site, direct URL fetching can fail on some sites due to browser cross-origin rules. When that happens, paste the page text or HTML into the big box and the analyzer still works.
