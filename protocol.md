# Civic Optics Protocol (COP‑1) – Scoring Standard

The Civic Optics Protocol evaluates civic information using five criteria.  Each criterion is scored on a 0–10 scale, where **0** means the site fails completely and **10** means it fully meets the criterion.  The Civic Optics Index (COI) is a weighted average of these scores.

| Criterion | Weight | Description |
| --- | ---: | --- |
| **Legibility** | 20 % | Measures whether the text and design are easy to read.  Consider font size, contrast, use of headings, bullet lists, and avoidance of dense paragraphs. |
| **Understandability** | 30 % | Judges whether the content uses plain language, avoids jargon, and explains necessary background.  Instructions should be clear and unambiguous. |
| **Navigability** | 20 % | Assesses how easily a user can find relevant information.  Well‑structured pages with clear sections, anchors, and links to external forms or calendars score higher. |
| **Currency** | 15 % | Reflects whether the information is up‑to‑date.  Pages should display a recent update date and match current legal deadlines or procedures. |
| **Actionability** | 15 % | Measures whether the page provides all the steps and links needed to complete the task (e.g., fill out a form, submit an application, contact the right office). |

## Scoring guidance

* **9–10:** Exemplary – fully satisfies the criterion.  For example, clear headings, large fonts, high‑contrast colors and bullet lists for legibility; plain‑English instructions with no missing steps for understandability.
* **7–8:** Strong – minor issues but largely meets expectations.
* **5–6:** Adequate – meets minimum requirements but room for improvement.
* **3–4:** Weak – significant obstacles to use (e.g., dense legalistic language or confusing navigation).
* **0–2:** Poor – fails to address the criterion.

## Calculating the Civic Optics Index

The COI is calculated as the weighted sum of the five criterion scores.  If **L** is the legibility score, **U** the understandability score, **N** the navigability score, **C** the currency score and **A** the actionability score, then:

```
COI = 0.20 × L + 0.30 × U + 0.20 × N + 0.15 × C + 0.15 × A
```

The final index can be expressed on a 0‑100 scale by multiplying the result by 10.
